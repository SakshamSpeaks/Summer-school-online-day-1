 Basic Structure
<!DOCTYPE html> – Declares HTML5 document.

<html lang="en"> – Root element; sets language to English.

<head> – Contains meta, title, and head info.

<meta charset="UTF-8"> – Sets character encoding.

<meta name="viewport"...> – Makes layout responsive on all devices.

<title> – Sets browser tab title.

<body> – Holds all the visible content on the page.

Text and Headings
<h1> to <h2> – Headings (main to sub-sections).

<p> – Paragraph text.

<br> – Line break (new line).

Navigation / Links
<a href="..."> – Creates clickable link to other pages or external sites.

Lists
<ul> – Unordered (bullet) list.

<ol> – Ordered (numbered) list.

<li> – List item in ul or ol.

Image
<img src="..." alt="..."> – Displays an image with alternative text.

Table
<table> – Creates a table.

<thead> – Header section of the table.

<tbody> – Body/content section of the table.

<tr> – Table row.

<th> – Header cell.

<td> – Table data cell.

 Form Inputs
<form> – Starts a form for user input.

<label> – Labels input fields.

<input type="text"> – Text input (for name).

<input type="email"> – Email input.

<input type="number"> – Number input (e.g., phone).

<input type="tel"> – Telephone input.

<input type="date"> – Date picker.

<input type="checkbox"> – Select multiple options.

<input type="radio"> – Select one option from a group.

<input type="range"> – Range/slider input.

<select> – Dropdown menu.

<option> – Option inside dropdown.

<textarea> – Multiline text input (for messages).

<button type="submit"> – Submit form button.

<output> – Shows output from range input.
